Cross domain and security

The same-origin policy restricts how a document or script loaded from one origin can interact with a resource from another origin. Two pages have the same origin if the protocol, port (if one is specified), and host are the same for both pages (iframe, ajax, etc.).

Http vs https is also important thing to mention.
